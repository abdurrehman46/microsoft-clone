# 🖥️ Microsoft Website Clone

This is a frontend clone of the Microsoft homepage created using HTML and CSS.  
Microsoft is one of the world's largest technology companies, known for products like Windows, Microsoft Office, Azure Cloud, and more.

🚀 Features  
Microsoft-style navigation bar  
Logo placement and menu alignment  
Left and Right Navbar sections  
Product cards section  
Styled buttons  
Clean UI layout  
Hover effects on links  

🛠️ Technologies Used  
HTML  
CSS  
Flexbox  

🏢 About Microsoft  
Microsoft is a global technology company founded in 1975.  
It develops software, hardware, and cloud services used worldwide.  
Some of Microsoft's popular products include:  

- Windows Operating System  
- Microsoft Office  
- Microsoft Edge Browser  
- Azure Cloud Services
